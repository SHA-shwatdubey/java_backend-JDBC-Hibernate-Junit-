EMPLOYEE SERVICE - Complete Details:
=====================================
Package: pom.capgemini.service.impl.EmployeeServiceImpl
Implements: EmployeeService

Methods:
  1. void addEmployee(Employee employee)
     - Validates employee data
     - Checks name, email, salary
     - Calls employeeDAO.insert()
     - Throws InvalidDataException on failure

  2. void updateEmployee(Employee employee)
     - Validates employee ID exists
     - Checks name format
     - Calls employeeDAO.update()
     - Throws ResourceNotFoundException if ID not found

  3. void removeEmployee(Integer id)
     - Validates ID (positive)
     - Checks if employee exists
     - Calls employeeDAO.delete()
     - Throws ResourceNotFoundException if not found

  4. Employee getEmployeeById(Integer id)
     - Validates ID
     - Calls employeeDAO.findById()
     - Throws ResourceNotFoundException if not found
     - Returns Employee object

  5. List<Employee> getAllEmployees()
     - Calls employeeDAO.findAll()
     - Returns all employees from database

  6. List<Employee> getEmployeesByDepartment(String department)
     - Validates department parameter
     - Calls employeeDAO.findByDepartment()
     - Returns filtered list

Annotations:
  • @Service (Spring stereotype)
  • @Autowired (Dependency injection)

Validation Rules:
  • Name: Required, minimum 2 chars
  • Email: Required, valid format
  • Salary: Required, >= 0
