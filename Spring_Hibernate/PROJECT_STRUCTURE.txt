LAYERS:
=======
1. Controller Layer (EmployeeController)
   - Handles Employee CRUD operations
   - Interacts with Service Layer

2. Service Layer (EmployeeService/EmployeeServiceImpl)
   - Business Logic Implementation
   - Data Validation
   - Exception Handling
   - Interacts with DAO Layer

3. DAO Layer (EmployeeDAO/EmployeeDao)
   - Data Access Operations
   - JPQL Queries
   - EntityManager Management
   - Database Interaction

4. Entity Layer (Employee, User)
   - JPA Entities
   - Table Mapping
   - Column Definitions

CONFIGURATION:
===============
- Spring Context: XML-based configuration
- Database: PostgreSQL
- ORM: Hibernate with JPA
- Transaction Manager: JPA Transaction Manager
- EntityManager: LocalContainerEntityManagerFactoryBean

KEY FEATURES:
=============
✓ Employee CRUD Operations (Create, Read, Update, Delete)
✓ Find Employees by Department
✓ Transaction Management with @Transactional
✓ Custom Exception Handling
✓ Data Validation
✓ Logging Utilities
✓ Component Scanning
✓ Auto-wiring with @Autowired
